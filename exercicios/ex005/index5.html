PRIMEIROS TESTES

NEGRITO

existem duas formas de colocar textos em negrito, por exemplo esse texto usando a tag <b>, no qual é uma tag não-semântica.

Agora, nesta frase, tenho um texto em destaque usando a tag <strong> que é semântica.

ITALICO

existe uma maneira de deixar o texto em italico usando a <i> que é não-semantica.

desta vez, temos um texto em enfase usando <em> que é semantica

Textos marcados

também podemos marcar textos como se um marcatexto tivesse sido usado usando a tag <mark>

Tamanho de textos

podemos mudar o tamanho dos textos usando a tag <small> para textos pequenos, entretando para textos grandes é apenas css

texto deletado e inserido

podemos marcar textos excluidos que podem ser lidos mas nao devem ser considerados usando a tag<del>

e marcar como inserido para enfase mostrando que foi colocado depois usando a tag <ins>

textos sobrescritos

podemos colocar textos sobrescritos usando a tag <sup>. por exemplo: x2

textos subscritos

podemos tambem colocar textos abaixo da escritura usando a tag <sub>. por exemplo: H20
